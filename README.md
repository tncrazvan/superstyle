# What is this?

This is a proof of concept of how to use SCSS as a template system to render websites on the server.


Run with
```bash
composer prod:start
```

# Debugging

- Start your XDebug 3 listener
- Set your breakpoints
- Run
  ```bash
  composer dev:start
  ```
